Teamgeist

Angular v1.5.8, Material v1.1.1, Express, Electron, MySQL or Postgres

The project was developed with analogy to the Upwork time counter. The REST Express server which was used here is common one written on PostgreSQL and MySQL. The web interface was written with the help of Angular-material. It uploads dynamicaly in depend of user (for example manger or developer). Developer has ability to review his activity statistic only. But the manager has ability to operate the projects and developer's porfiles. The client side was created with the Electron and its aim is to send Screenshot with all required data to the server.
