MWO Bugtracker

Codeigniter, Bootstrap 3, MySQL

Bugtracker for the Metal War Online game. Accroding to the name of project it was created to coordinate QA department. There are four roles with its own rights and accesses to the application's functionality. The application was written with the help of Codeigniter for backend and Bootstrap for the frontend.
